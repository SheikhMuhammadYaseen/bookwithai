# Physical AI & Humanoid Robotics - Digital Book

This project is an interactive learning platform focused on Physical AI and Humanoid Robotics, built using Docusaurus.

## About

This digital book serves as a comprehensive resource for learning about the exciting field of Physical AI and its application in humanoid robotics. It covers a wide range of topics from the basics of sensors and actuators to advanced concepts in AI-driven perception and control.

## Technologies Used

-   **Docusaurus:** A modern static website generator for building documentation websites.
-   **React:** A JavaScript library for building user interfaces.
-   **TypeScript:** A typed superset of JavaScript that compiles to plain JavaScript.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

-   Node.js (version >=20.0)
-   npm

### Installation & Running

1.  Navigate to the `digital-book` directory:
    ```sh
    cd digital-book
    ```
2.  Install NPM packages:
    ```sh
    npm install
    ```
3.  Start the development server:
    ```sh
    npm start
    ```

The site will be available at `http://localhost:3000`.

## Building the Project

To create a static production build, run the following command from the `digital-book` directory:

```sh
npm run build
```

The build artifacts will be stored in the `build` directory.

# Chapter 6: Gazebo Simulation

## 6.1 Introduction to Gazebo {#ch6-sec1-intro-to-gazebo}

Gazebo is a powerful 3D robotics simulator that allows you to test your robot's algorithms, sensors, and actuators in a virtual environment. It is a a physics-based simulator that can accurately model the real world. Gazebo is an essential tool for robotics development, as it allows you to test your code without having to risk damaging a physical robot.
